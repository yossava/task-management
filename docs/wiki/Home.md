## What is ProjectHub?

ProjectHub is a modern, feature-rich project management application built with:
- **Next.js 15** - React framework with app router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **LocalStorage** - Client-side data persistence
- **PWA** - Offline-first progressive web app

## Key Features

### Task Management
- Create, edit, and organize tasks
- Drag-and-drop Kanban boards
- List and calendar views
- Priority levels and progress tracking
- Rich text descriptions with Markdown

### Advanced Features
- **Subtasks** - Break down complex tasks
- **Dependencies** - Manage task relationships
- **Time Tracking** - Estimate and track time
- **Comments** - Collaborate on tasks
- **Tags** - Categorize and filter
- **Search** - Powerful filtering and saved searches

### Productivity Tools
- **Command Palette** - Keyboard-driven navigation (Cmd+K)
- **Templates** - Reusable task configurations
- **Recurring Tasks** - Automated task creation
- **Dashboard** - Analytics and insights
- **Offline Mode** - Work without internet
