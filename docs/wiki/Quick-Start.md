# Quick Start Guide

Get up and running with ProjectHub in 5 minutes.

## 1. Access the App

Open ProjectHub in your browser:
```
http://localhost:3000
```

## 2. Create Your First Board (30 seconds)

1. Click the **"New Board"** button
2. Type a name: "My First Project"
3. Press **Enter**

âœ… Your board is created!

## 3. Add Your First Task (30 seconds)

1. Click **"+ Add Task"** in the "To Do" column
2. Type: "Review project requirements"
3. Press **Enter**

âœ… Your first task is created!

## 4. Organize Your Task (1 minute)

1. Click on the task to open details
2. Add a description (optional)
3. Set priority to **High**
4. Add a due date (click calendar icon)
5. Add tags: "planning", "priority"
6. Click outside to close

âœ… Task is now organized!

## 5. Try Advanced Features (2 minutes)

### Add a Subtask
1. Open the task
2. Scroll to "Subtasks" section
3. Type: "Read documentation"
4. Click **Add**

### Start Time Tracking
1. In the same task
2. Find "Time Tracking" section
3. Click **"Start Timer"**
4. Timer begins counting

### Use Command Palette
1. Press **Cmd+K** (or **Ctrl+K** on Windows)
2. Type "new task"
3. Press **Enter**
4. Create another task instantly!

## 6. Explore Views (30 seconds)

Click the view buttons at the top:
- **Board View** - Kanban columns
- **List View** - Compact list
- **Calendar View** - Date-based

## 7. Check Dashboard (30 seconds)

1. Click **"Dashboard"** in navigation
2. See your task statistics
3. View recent activity
4. Check upcoming deadlines
